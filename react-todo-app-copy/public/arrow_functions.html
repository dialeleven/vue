Demo of JavaScript ES6 arrow functions. Open the console (F12) to see the output. 
   1. Move the Parentheses

   function definition: function () {}
   arrow function: () => {}

   // original function syntax
   function () {}

   // move the parentheses
   () function {}

   // change `function` to `=>`
   () => {}
   
   Those parentheses are the same old function parentheses and you can think of them as your guide. 
   The braces {} still contain the contents of the function.
